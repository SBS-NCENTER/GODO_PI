# PROGRESS/ — Weekly session log archives

ISO 8601 weeks (Mon–Sun KST). Each archive holds the session-log
entries from its calendar week. The master file `../PROGRESS.md`
keeps the live project state (current phase, decisions, open
questions, quick reference) and the Index of weekly archives.
Rotation rule: at session-close, the previous "most recent" session
block in the master rotates into its matching weekly file.
